Screen Recorder

A proof of concept screen recorder program written in C#. The program writes video to capture.mp4 using MPEG4 video codec.

Github Repository: https://github.com/rkim3-emich/ScreenRecorder

Technologies Used:

  • C#
  • Windows Presentation Foundation (WPF)
  • AForge.NET
  • FFMPEG
