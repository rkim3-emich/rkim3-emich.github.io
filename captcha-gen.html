Captcha Generator

This is a simple captcha generator that combines handwritten digits from the MNIST handwritten digit dataset. Practically, more secure methods of captcha should be used to deter bots due to the increasing capabilities of bots especially as machine learning allows for more advanced image recognition. Furthermore, the program renders the captcha and answer on the client side due to constraints of github pages which should not done due to the risk of client side manipulation of the code. For more secure captcha generation, the code and answer should be generated on the server with the captcha being sent to the user and then the user answer should be sent to the server to be compared against the server's answer.
